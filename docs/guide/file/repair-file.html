# 修复文件

TIP

修复文件是将文件与游戏原文件对比,补充缺失的贴图,主要用于修复红叉

选择需要修复的文件,支持多选

点击左侧文件列表右键菜单中的修复文件

WARNING

如果未能修复成功。请确认文件的路径及程序设置中的游戏目录是否正确
