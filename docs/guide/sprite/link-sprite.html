    # 索引贴图

    TIP

    索引贴图只有一个指向其他贴图下标的索引。所有的数据都来自于被索引的贴图。

    WARNING

    索引贴图指向的贴图只在同文件下有效

    # 将贴图链接到其他贴图上

    点击右侧贴图列表右键菜单中的索引贴图

    输入被索引贴图的下标。只能为数字,不可以小于0和大于贴图总数
