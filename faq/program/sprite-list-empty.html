# 贴图列表为空

此类文件一般经过1.6以下版本ES或其他软件加密

TIP

ExtractorSharp 1.6版本后不再支持加密

为了保护已加密文件,1.6版本以后对于此类文件进行清空贴图列表的处理

WARNING

打开此类文件后请不要进行保存操作!
